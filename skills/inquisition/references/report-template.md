# Report Template

Present findings before the verdict. Use this section order:

1. **Cross-Referenced Findings** — items confirmed by both passes or sharing a root cause
2. **Architecture-Only Findings** — from Pass A only
3. **Compliance-Only Findings** — from Pass B only
4. **Prioritized Action Items** — ranked list with Source, Category, Location, Why it matters, Suggested fix direction
5. **Residual Risk and Test Gaps** — unverified areas, missing checks, scope limitations
6. **Combined Verdict** — short synthesis after all findings

Each finding must cite `file:line` or command output. Use severity levels from [`severity.md`](severity.md).
